### Описание настроек:
Настраивается приложение в appsettings
 - Folder - папка в которой содержится файл экспортированный из Money Flow. Название файла должно быть Money_Flow.csv
 - CategoriesIncome - массив категорий дохода
 - CategoriesNecessarily - категории для бюджета, здесь перечисляем обязательные категории
 - Tags - набор тэгов которые хотим исключать при просмотре статистики
 - TotalBudgetNecessarily - сумма бюджета на обязательные траты в месяц
 - TotalBudgetNotNecessarily - сумма бюджета на необязательные траты в месяц
